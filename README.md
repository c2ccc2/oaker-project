## 平台介绍
   九橡项目工时管理系统（oak project），是一款轻量级项目工时统计系统，可通过员工工时上报的方式，来记录项目所花费的工时，帮助企业进行项目工时统计、核算人工成本。
实时、动态、真实的展示项目投入。

## 业务特色
1、视角创新
    从管理视角设计系统功能，实时在线收集和分析项目工时数据。
2、功能创新
    围绕项目工时统计，延展出任务看板、原型管理、知识库等项目实用功能。

## 管理创新
   用数据说话，智能统计各个项目工时情况、自动进行项目风险提醒。
   
## 应用创新
   开源、免费、迭代快、灵活性强。
